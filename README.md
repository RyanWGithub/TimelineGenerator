一个基于 React 的现代化时间轴生成工具，帮助你记录、可视化和管理重要的历史时刻或个人回忆。

## ✨ 主要功能

- **交互式时间轴**：支持无限水平滚动、鼠标滚轮缩放、拖拽移动以及键盘方向键导航。
- **丰富的事件管理**：
  - 支持添加标题、详细描述、日期。
  - 支持上传图片（包括 GIF 动图）。
  - 自动生成事件序号，支持根据时间自动排序。
- **关联关系可视化**：
  - 支持建立事件之间的关联。
  - 自动绘制双向贝塞尔曲线连接相关事件。
  - 在连线两端显示关联事件的标题、序号以及时间间隔天数。
  - 支持一键开启/关闭连线显示。
- **强大的搜索功能**：
  - 支持按标题、描述内容或日期进行搜索。
  - 搜索结果支持点击快速跳转定位。
- **本地数据存储**：
  - 使用 IndexedDB (idb) 进行本地存储，数据安全且持久化。
  - 无需后端服务，打开浏览器即可使用。
- **导入与导出**：
  - 支持将时间轴数据导出为 JSON 备份。
  - 支持从备份文件恢复数据。
